# 🤖 AI-Powered Figma → Jira Automation with MCP

An AI workflow demonstrating how Product Managers can convert product designs into structured Jira delivery artifacts using MCP (Model Context Protocol), Claude Desktop, Figma, and Jira Cloud.

The project explores how GenAI can reduce repetitive product operations and accelerate the transition from design to execution.

---

## 💡 Description & Goal

Translating product designs into Jira epics and user stories is often a manual and time-consuming process.

The goal of this project was to explore how AI agents and MCP integrations can automate this workflow by transforming Figma designs directly into Jira-ready artifacts with minimal manual effort.

This exercise focuses on practical GenAI usage for Product Managers and demonstrates how AI can support product discovery and delivery workflows.

---

## 💡 Key Features

- Claude Desktop setup and MCP configuration
- Direct connection with Jira Cloud
- Native Figma connector integration
- AI-generated Jira Epics and User Stories from designs
- Prompt-driven workflow execution
- End-to-end automation without manual ticket creation
- Hands-on exploration of AI-assisted product operations

---

## 🛠 Tech Stack

- Claude Desktop
- MCP (Model Context Protocol)
- Jira Cloud
- Figma Connector
- LLM prompting
- AI Agent workflow patterns

---

## ⚙️ Workflow Overview

### 1. Environment Setup
- Install Claude Desktop
- Configure MCP

### 2. Tool Connections
- Connect Jira Cloud
- Connect Figma through the built-in connector

### 3. AI Execution
- Provide prompt
- AI interprets design context
- Generates Jira Epics and User Stories automatically

---

## 🧠 Product Thinking

- Reduces operational overhead for Product Managers
- Accelerates design → delivery handoffs
- Demonstrates practical GenAI adoption in product workflows
- Explores human + AI collaboration patterns
- Evaluates AI capability beyond content generation toward workflow execution
